
# Alumni Management System

A comprehensive SaaS platform for managing alumni data across universities.

## Overview

Alumni Management System is a subscription-based solution designed to help universities efficiently manage, track, and engage with their alumni community.

## Features

- **Alumni Database Management** - Centralized storage for alumni information
- **Profile Management** - Customizable alumni profiles and records
- **Event Management** - Organize and track alumni events
- **Communication Tools** - End to end message and group message.
<!-- - **Analytics & Reporting** - Insights on alumni engagement and demographics -->
- **Multi-University Support** - Manage multiple institutions from one platform

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- php 8+
- Database (MySQL/PostgreSQL)

### Installation

```bash
git clone <repository-url>
cd alumni-management
npm install
```

### Configuration

Create a `.env` file with required environment variables.

### Running the Application

```bash
npm run build
php artisan serve
```
